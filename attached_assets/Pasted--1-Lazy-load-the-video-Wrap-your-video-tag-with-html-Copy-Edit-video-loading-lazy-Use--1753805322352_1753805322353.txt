🛠 2. Use async/defer on scripts

You're bundling with Vite — confirm in your final HTML (inside dist/public/index.html) that <script> tags are using type="module" and are at the bottom of <body>.

🛠 3. Enable gzip or Brotli compression

In express, add this (you already have the package):

ts
Copy
Edit
import compression from 'compression';
app.use(compression());
🛠 4. Enable long-term browser caching

In express.static(), set cache headers:

ts
Copy
Edit
app.use(express.static('public', {
  maxAge: '1y',
  etag: false
}));
